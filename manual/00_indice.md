# Manual do Usuário — SIGE

Bem-vindo ao Manual do Usuário do **SIGE** (Sistema Integrado de Gestão Empresarial).

Este manual reúne, em um único lugar, as instruções de uso de **todos os módulos** do SIGE. Ele é vivo: cada módulo terá seu capítulo revisado e expandido conforme as auditorias de uso forem concluídas.

## Como usar este manual

- Use o **sumário lateral** para saltar para um capítulo.
- Cada capítulo descreve **para que serve** o módulo, **como usá-lo no dia a dia** e os **principais cuidados** a tomar.
- Você pode **baixar o manual completo** clicando em "Baixar manual" no topo da página de visualização. O arquivo abre como uma página de impressão pronta para "Salvar como PDF" pelo navegador.

## Estrutura prevista

1. **Introdução** — visão geral do sistema, para quem é, como navegar.
2. **Módulos** — um capítulo por módulo do SIGE:
   - Dashboard
   - Funcionários
   - Obras
   - RDO (Relatório Diário de Obra)
   - Cronograma
   - Financeiro
   - Propostas Comerciais
   - Folha de Pagamento
   - CRM de Leads
   - Métricas
   - Catálogo (Insumos e Serviços)
   - Estoque / Almoxarifado
   - Frota
   - Alimentação
   - Portal do Cliente

> Os capítulos marcados como **"Em construção"** ainda não foram preenchidos. Eles serão adicionados nas próximas atualizações.
