Sistema de Carrinho

Agora você pode adicionar múltiplos itens ao carrinho:

  • Selecione o funcionário e obra primeiro
  • Adicione vários itens ao carrinho
  • Todos os itens terão o mesmo funcionário/obra
  • Finalize a saída de uma vez
Itens Serializados

Para itens com controle por número de série:

  • Selecione os itens específicos
  • Cada item será vinculado ao funcionário
  • Status muda para "EM USO"
Itens Consumíveis

Para itens de controle por quantidade:

  • Informe a quantidade que será retirada
  • Sistema usa FIFO
  • Quantidade é deduzida do estoque
Importante: Todos os itens do carrinho terão o mesmo funcionário e obra.
