Como Funciona?
Itens Serializados

Para itens com controle por número de série:

  • Selecione os itens específicos que serão entregues
  • Cada item será vinculado ao funcionário
  • Status do item muda para "EM USO"
  • Permite rastreamento individual
Itens Consumíveis

Para itens de controle por quantidade:

  • Informe a quantidade que será retirada
  • Sistema usa FIFO (primeiro a entrar, primeiro a sair)
  • Quantidade é deduzida do estoque disponível
  • Se estoque zerar, status muda para "CONSUMIDO"
Importante: O funcionário é obrigatório para todas as saídas. A obra é opcional.
