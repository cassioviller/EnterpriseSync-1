 Auditoria de Cálculo: Insumos / Catálogo / Orçamento

 Context

 O usuário reportou que os valores exibidos no Serviço do Catálogo não batem com os valores
 exibidos no Orçamento. A auditoria identificou inconsistência semântica em como preco_unitario
 é tratado em dois contextos distintos:

 - Catálogo (calcular_precos_servico em orcamento_service.py): trata preco_unitario como preço
 técnico unitário = preco_vigente / fator_comercial
 - Snapshot do Orçamento (snapshot_from_servico em orcamento_view_service.py): armazena
 preco_unitario = preco_vigente() (preço da embalagem — ERRADO)

 Isso causa divergência na exibição de custo_unit_tec e subtotal_unitario, mas não corrompe
 custo_total nem venda_total (já corrigidos pela Task #47).

 Bugs confirmados

 Bug 1 — snapshot_from_servico() em orcamento_view_service.py ~linhas 40-50
 # ATUAL (errado)
 preco_unitario = insumo.preco_vigente()        # preço da embalagem
 subtotal_unitario = comp.coeficiente * preco   # coef × pkg_price

 # CORRETO
 fator = insumo.fator_comercial or 1
 preco_tecnico = insumo.preco_vigente() / fator  # preço técnico unitário
 subtotal_unitario = comp.coeficiente * preco_tecnico
 # + adicionar campo preco_embalagem = insumo.preco_vigente()

 Bug 2 — recalcular_item() em orcamento_view_service.py ~linhas 90-100
 # ATUAL (errado)
 sub_unit = comp['coeficiente'] * preco   # preco = preco_vigente (embalagem)

 # CORRETO
 fator = comp.get('fator_comercial', 1) or 1
 sub_unit = comp['coeficiente'] * preco / fator  # preco técnico unitário
 # Fallback legacy: se fator não presente no snapshot, usar 1 (manter retrocompatibilidade)

 Bug 3 — JS em templates/orcamentos/editar.html ~linhas 1000-1015
 // ATUAL (errado) — usa custo técnico para totalizar
 custoTot = custoUnit * qtd;   // custo técnico × qty

 // CORRETO — usar custo de compra real para totais (alinhado com backend)
 custoTot = custoCompra;       // já acumulado como custo real de compra

 Bug 4 — JS picker de insumo em editar.html ~linha 900 (approx)
 // ATUAL (suspeito)
 comp_preco_unitario = insumo.preco_vigente   // embalagem

 // CORRETO
 comp_preco_unitario = insumo.preco_vigente / (insumo.fator_comercial || 1)
