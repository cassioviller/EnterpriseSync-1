# Prompt: Manual do Módulo Funcionários - EnterpriseSync

## 🎯 Objetivo

Criar o **Capítulo 3** do manual de uso do sistema EnterpriseSync, focado no **Módulo de Gestão de Funcionários**.

## 📝 Estrutura do Manual

O manual deve ser escrito em **Markdown**, com linguagem clara, objetiva e profissional. Use tabelas, listas numeradas e imagens (placeholders) para facilitar o entendimento.

### **Capítulo 3: Módulo de Gestão de Funcionários**

#### **3.1. Introdução à Gestão de Funcionários**
- Visão geral do módulo e sua importância para o RH e gestão de pessoal.
- Funcionalidades cobertas: cadastro, ponto, reconhecimento facial, relatórios.

#### **3.2. Tela Principal de Funcionários**
- Como acessar a tela de listagem de funcionários.
- Explicação da tabela de funcionários (colunas, filtros, busca).
- Ações rápidas (editar, desativar, ver perfil).

#### **3.3. Cadastrando um Novo Funcionário**
- Passo a passo detalhado para cadastrar um novo funcionário.
- **3.3.1. Dados Pessoais**
  - Preenchimento de nome, CPF, RG, data de nascimento, etc.
- **3.3.2. Dados Contratuais**
  - Vinculação a departamento e função.
  - Definição de salário e tipo de contrato.
  - Vinculação a horário de trabalho.
- **3.3.3. Dados de Acesso**
  - Como criar um usuário de sistema para o funcionário.
  - Definição de permissões de acesso.

#### **3.4. Perfil do Funcionário**
- Como acessar o perfil completo de um funcionário.
- Explicação de cada seção do perfil:
  - Informações pessoais
  - Histórico de ponto
  - Histórico de alocação em obras
  - Documentos
  - Dashboard individual

#### **3.5. Reconhecimento Facial**
- **3.5.1. Importância do Reconhecimento Facial**
  - Segurança no controle de ponto.
  - Agilidade no registro.
- **3.5.2. Cadastrando Fotos Faciais**
  - Passo a passo para cadastrar as fotos de um funcionário.
  - Recomendações para fotos de qualidade (iluminação, fundo, etc.).
  - Quantidade ideal de fotos para o cadastro.

#### **3.6. Controle de Ponto Eletrônico**
- **3.6.1. Registrando o Ponto**
  - Como o funcionário registra o ponto (entrada/saída) usando reconhecimento facial.
  - Como o sistema valida a identidade.
- **3.6.2. Visualizando Registros de Ponto**
  - Como o gestor visualiza os registros de ponto de sua equipe.
  - Explicação da tela de controle de ponto (horários, atrasos, horas extras).
- **3.6.3. Editando e Justificando Registros**
  - Como editar um registro de ponto (com permissão).
  - Como adicionar justificativas para atrasos ou faltas.

#### **3.7. Relatórios de Funcionários**
- Como acessar a área de relatórios de funcionários.
- Tipos de relatórios disponíveis:
  - Lista de funcionários ativos
  - Relatório de aniversariantes
  - Relatório de folha de ponto
  - Relatório de absenteísmo
- Como gerar e exportar relatórios (PDF/Excel).

## 🎨 Estilo e Tom

- **Tom**: Profissional, didático e acessível.
- **Linguagem**: Evitar jargões técnicos sempre que possível.
- **Formato**: Markdown, com uso de títulos, subtítulos, negrito, tabelas e listas.
- **Imagens**: Usar placeholders como `[IMAGEM: Tela de listagem de funcionários]` ou `[IMAGEM: Cadastro de fotos faciais]` para indicar onde as imagens devem ser inseridas.

## 🚀 Ação

Com base neste prompt, gere o arquivo `03_manual_funcionarios.md` com o conteúdo completo do Capítulo 3.
