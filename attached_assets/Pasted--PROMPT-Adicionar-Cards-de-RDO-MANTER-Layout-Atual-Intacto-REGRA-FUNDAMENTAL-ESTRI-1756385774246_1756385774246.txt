### Estrutura de Cada Card:
```

│  GALPÃO INDUSTRIAL ESTRUTURAS METÁLICAS     │  ← Nome da obra (NEGRITO)
│  29/08/2025                                 │  ← Data

```

### Especificações do Card:

**1. NOME DA OBRA (Topo)**
- Texto grande e **negrito**
- Cor: #2d5a27 (verde escuro)
- Tamanho: 1.3rem
- Font-weight: 700

**2. DATA (Segunda linha)**
- Texto menor
- Cor: #6c757d (cinza)
- Tamanho: 0.9rem
- Formato: DD/MM/AAAA

**3. BARRA DE PROGRESSO (Centro)**
- Altura: 8px
- Cor da barra: Gradiente verde (#28a745 → #20c997)
- Fundo: #e9ecef
- Bordas arredondadas
- Porcentagem ao lado: 1.2rem, negrito, cor #2d5a27

**4. FUNCIONÁRIO (Inferior)**
- Ícone: 👤 ou <i class="fas fa-user"></i>
- Cor: #495057
- Tamanho: 0.95rem

**5. BOTÕES (Rodapé)**
- "Ver Detalhes": background #28a745, cor branca
- "Editar RDO": background #007bff, cor branca
- Padding: 0.5rem 1rem
- Border-radius: 6px

### CSS do Card:
```css
.rdo-card {
    background: white;

    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);

```

### Grid dos Cards:
- **Desktop:** 3 cards por linha
- **Tablet:** 2 cards por linha
- **Mobile:** 1 card por linha
- **Gap:** 1.5rem entre cards
