# PROMPT 0: Versionamento do Sistema (V1 vs V2) por Tenant

**Objetivo:** Implementar um sistema de versionamento (Feature Flag) no nível do Administrador (Tenant). O SuperAdmin poderá escolher se um novo Admin usará a versão clássica (V1) ou a nova versão integrada (V2). Isso garante que os clientes atuais não sejam afetados pelas novas mudanças.

**Arquivos a serem modificados:**
- `models.py`
- `views/admin.py`
- `templates/super_admin_dashboard.html`
- `utils/tenant.py` (ou `helpers.py`)
- `templates/base.html`

Por favor, implemente as seguintes alterações passo a passo:

## Passo 1: Atualização do Modelo Usuario
No arquivo `models.py`, localize a classe `Usuario` e adicione um novo campo para controlar a versão do sistema do tenant:
```python
# Adicionar na classe Usuario
versao_sistema = db.Column(db.String(10), default='v1', nullable=False) # 'v1' ou 'v2'
```

## Passo 2: Atualização do SuperAdmin Dashboard (Backend)
No arquivo `views/admin.py`, localize a rota `@main_bp.route('/super-admin/criar-admin', methods=['POST'])` (função `criar_admin`):
1. Capture o novo campo do formulário: `versao_sistema = request.form.get('versao_sistema', 'v1')`
2. Adicione este campo na criação do objeto `novo_admin`:

novo_admin = Usuario(

## Passo 4: Helper de Verificação de Versão
Crie uma função utilitária (pode ser em `utils/tenant.py` ou `helpers.py`) para verificar facilmente a versão do admin atual:
```python
def is_v2_active():
    """Verifica se o tenant atual está usando a V2 do sistema"""
    from flask_login import current_user
    from models import Usuario, TipoUsuario
    
    if not current_user.is_authenticated:
        return False
        
    # Se for o próprio admin
    if current_user.tipo_usuario == TipoUsuario.ADMIN:
        return current_user.versao_sistema == 'v2'
        
    # Se for funcionário, busca o admin dele
    if current_user.admin_id:
        admin = Usuario.query.get(current_user.admin_id)
        return admin and admin.versao_sistema == 'v2'
        
    return False
```
*Nota: Exporte esta função para o contexto do Jinja (templates) no `main.py` ou `__init__.py` usando `@app.context_processor` para poder usar `{% if is_v2_active() %}` nos templates HTML.*
