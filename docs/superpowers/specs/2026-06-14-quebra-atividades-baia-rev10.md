## Sequência macro (cronograma)
`Fundação (infra embutida → marco pré-concretagem → concreto → cura) → Corredores` ∥
`LSF (painelização→verticalização→contraventamento) ∥ pilares roliços ∥ fabricação telhado (oficina)` →
`montagem viga I → shingle` ; `fechamentos (lã→placa→junta→basecoat→pintura ∥ régua→stain) → pedra moledo, forro` ;
`portão/cercado (oficina ∥ in loco) → stains` ; `elétrica em paralelo (infra na fundação … pontos pós-cobertura)`.
